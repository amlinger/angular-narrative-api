Cache
service in module api.narrative

Description

A cache object used to store and retrieve data.

Methods

  • get(key)

    Gets and entry from the cache.

    Parameters
    ParamTypeDetails
    keystring

    the key under which the cached data is stored.

    Returns
    *

    the value stored, or nullif the value does not exist.

  • put(key, value)

    Adds entries in the cache.

    Parameters
    ParamTypeDetails
    keystring

    the key under which the cached data is stored.

    value*

    the value to store alongside the key. If it is null or undefined, the key will not be stored.

    Returns
    *

    the value stored.

  • remove(key)

    Removes an entry from the cache.

    Parameters
    ParamTypeDetails
    keystring

    the key under which the cached data is stored.

    Returns
    *

    the value stored.

  • removeAll()

    Removes all entries from the cache.
