NarrativeRequest
service in module api.narrative

Description

A method for making authorized requests with parameters to Narratives API on Narratives Open Platform.

Dependencies

  • $http

Usage

NarrativeRequest(method, url[, parameters][, authOrConfig]);

Parameters

ParamTypeDetails
methodstring

The HTTP method to be used for the reqest.

urlstring

The URL relative to the supplied full API URL.

parameters
(optional)
object

Optional URL parameters to add to the request.

authOrConfig
(optional)
NarrativeAuthobject

Either a NarrativeAuth or a configuration object for this particular request. The configuraiton object accepts the same attributes as the default selection for NarrativeCacheProviders default values.

Returns

promise

Returns the promise produced by the $http service.
